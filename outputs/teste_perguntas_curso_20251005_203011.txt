PERGUNTA 4:
Quais são as medidas de tendência central (média, mediana)?

──────────────────────────────────────────────────────────────────────
RESPOSTA:
**Medidas de Tendência Central**

As medidas de tendência central são estatísticas que descrevem o valor central de uma distribuição de dados. Elas são importantes para entender a tendência ou tendência central de uma série de dados.

**Média**

A média é a medida de tendência central mais comum. Ela é calculada somando todos os valores da série de dados e dividindo o resultado pela quantidade de valores.

Fórmula: Média = (Soma dos valores) / Número de valores

Exemplo: Se você tiver os valores 2, 4, 6, 8, 10, a média seria (2 + 4 + 6 + 8 + 10) / 5 = 6.

**Mediana**

A mediana é a medida de tendência central que divide a série de dados em dois grupos iguais. Ela é calculada ordenando os valores da série de dados em ordem crescente e encontrando o valor do meio.

Exemplo: Se você tiver os valores 2, 4, 6, 8, 10, a mediana seria 6, pois é o valor do meio quando os valores são ordenados em ordem crescente.

**Importância**

As medidas de tendência central são importantes porque:

* Elas ajudam a entender a tendência central de uma série de dados.
* Elas são úteis para comparar diferentes séries de dados.
* Elas são fundamentais para a análise estatística e a tomada de decisões.

**Exemplo em CSV**

Suponha que você tenha um arquivo CSV com os seguintes dados:

| Valor |
| --- |
| 2 |
| 4 |
| 6 |
| 8 |
| 10 |

Para calcular a média e a mediana, você pode usar a fórmula acima ou utilizar uma ferramenta de análise de dados, como o Excel ou o Python.

**Conclusão**

As medidas de tendência central, como a média e a mediana, são fundamentais para entender a tendência central de uma série de dados. Elas são importantes para a análise estatística e a tomada de decisões. Além disso, elas são fáceis de calcular e entender, tornando-as uma ferramenta útil para qualquer análise de dados.

PERGUNTA 5:
Qual a variabilidade dos dados (desvio padrão, variância)?

──────────────────────────────────────────────────────────────────────
RESPOSTA:
**Análise de Variabilidade dos Dados**

Para calcular a variabilidade dos dados, precisamos calcular o desvio padrão e a variância.

**Desvio Padrão (DP)**

O desvio padrão é uma medida da dispersão dos dados em relação à média. Ele é calculado como a raiz quadrada da variância.

Fórmula: DP = √(Σ(xi - μ)^2 / (n - 1))

Onde:
- xi é cada valor individual dos dados
- μ é a média dos dados
- n é o número de dados
- Σ é a soma dos valores

**Variância**

A variância é uma medida da dispersão dos dados em relação à média. Ela é calculada como a média dos quadrados das diferenças entre cada valor individual e a média.

Fórmula: Variância = Σ(xi - μ)^2 / (n - 1)

Onde:
- xi é cada valor individual dos dados
- μ é a média dos dados
- n é o número de dados
- Σ é a soma dos valores

**Exemplo**

Suponha que tenhamos os seguintes dados:

| Valor |
| --- |
| 10 |
| 12 |
| 11 |
| 13 |
| 14 |

Média (μ) = (10 + 12 + 11 + 13 + 14) / 5 = 12

Desvio Padrão (DP) = √((10-12)^2 + (12-12)^2 + (11-12)^2 + (13-12)^2 + (14-12)^2) / (5 - 1)
= √(4 + 0 + 1 + 1 + 4) / 4
= √10 / 4
= 1,58

Variância = (4 + 0 + 1 + 1 + 4) / (5 - 1)
= 10 / 4
= 2,5

**Conclusão**

A variabilidade dos dados é uma medida importante para entender a dispersão dos dados em relação à média. O desvio padrão e a variância são medidas comuns utilizadas para avaliar a variabilidade. No exemplo acima, o desvio padrão é de 1,58 e a variância é de 2,5.
